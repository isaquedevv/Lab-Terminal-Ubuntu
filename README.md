# Projeto Web Via Terminal Ubuntu

Esse é um projeto prático desenvolvido para dominar a linha de comando Linux, focando na construção de uma estrutura web sem o use de interfaces gráficas.

---

## Comandos Utilizados
| Comandos | Funções |
| :---- | :---- |
| `pwd` | Para localizar meu diretório atual |
| `ls` | Lista arquivos e pastas
| `cd` | Entra em pastas
| `mkdir` | Para criar uma estrutura de pastas (`css`,`js`, `etc..`)
| `touch` | Cria arquivos (`txt`, `html`...)
| `cp` | Copia arquivos e pastas
| `mv` | Move ou renomeia arquivos e pastas
| `rm` | Remove/Deleta arquivos desejados

---

## Opiniões sobre os Editores
* **Nano:** Em minha opinião achei mais facil para editar arquivos rápidamente.
* **Vim:** Um pouco mais complicado, porém mais intuitivo, sendo relativamente facil de entender os comandos sendo digitados por causa de suas cores customizadas.

---

## Como visualizar
1. Clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
